# 🚀 Portfolio Website

Modern, animated portfolio showcasing projects, skills, and services. Built with React, TypeScript, and Vite for blazing-fast performance.

## ✨ Features

- **Smooth Animations** — Framer Motion for fluid micro-interactions
- **Responsive Design** — Mobile-first, works on all devices
- **Skills Dashboard** — Interactive skill cards with filtering and proficiency indicators
- **Dark Theme** — Custom design system with CSS variables
- **SEO Optimized** — Meta tags and semantic HTML

## 🛠️ Tech Stack

- **Frontend:** React 18, TypeScript
- **Styling:** Tailwind CSS + Custom CSS
- **Animations:** Framer Motion
- **Build:** Vite
- **UI Icons:** Lucide React
- **Deployment:** Vercel

## 📦 Installation

**Prerequisites:** Node.js 16+

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
