# Portfolio Website

This repository contains the portfolio website source for local development and deployment.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Build for Production

1. Build the project:
   `npm run build`
2. Preview the production build:
   `npm run preview`
